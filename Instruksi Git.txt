INSTRUKSI GIT
1) git pull
   Untuk dapat update an terbaru dari folder project
2) Edit file-file
   Pastikan tidak ada bug/error sebelum diupload
3) git add .
   Untuk menandai file yang akan dicommit (titik "." berarti semua file)
4) git commit -m "Komentar"
   Untuk merampungkan updatean
5) git push
   Untuk mengupload updatean

Jika konflik:
1) git pull
2) perbaiki konflik file
3) git add .
4) git commit -m "Komentar"
5) git push
